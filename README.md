# Rubella-model-VIMC-runs-2023-24
This repository holds the code and input files used for the rubella model runs for the Vaccine Impact Modelling Consortium (VIMC) during 2023-24 (referred to as the "UKHSA model" or, previously the "PHE model").  It also includes examples of output files.  Please read the file "Readme - rubella model overview.pdf" for further details about the files
